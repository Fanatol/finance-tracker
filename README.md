# 💰 Личные финансы — трекер доходов и расходов

Веб-приложение для учёта личных финансов. Работает полностью в браузере, данные хранятся в localStorage. Позволяет добавлять, редактировать и удалять транзакции, фильтровать по категории, типу и дате, просматривать баланс и статистику по категориям.

## 📌 Техническое задание

### Обязательные требования

1. **Добавление транзакции** с указанием:
   - тип (доход / расход)
   - сумма (положительное число)
   - категория (фиксированные списки)
   - дата (выбор из календаря, по умолчанию текущая)
   - описание (необязательно)

2. **Отображение баланса**

3. **Просмотр списка транзакций**

4. **Фильтрация транзакций**:
   - по категории
   - по типу (доходы / расходы / все)
   - по дате (день / неделя / месяц / произвольный период)

5. **Удаление транзакции** с подтверждением

6. **Редактирование транзакции** 

7. **Сохранение данных между сессиями** 

8. **Адаптивный интерфейс**

### Дополнительно реализованные требования

- Переключение трёх тем оформления (светлая, тёмная, old money) с сохранением выбора
- Экспорт всех транзакций в JSON
- Сброс данных к демо‑набору
- Статистика доходов и расходов по категориям

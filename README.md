# 💰 Личные финансы — трекер расходов и доходов

Веб-приложение для учёта личных финансов с возможностью добавления, редактирования и удаления транзакций, фильтрации и аналитики. Данные сохраняются в браузере (localStorage). Проект выполнен в рамках курсовой работы 1 курса программной инженерии.

## 📌 Функциональные требования (обязательные)

- ✅ Добавление транзакции с указанием типа (доход/расход), суммы, категории, даты и описания.
- ✅ Отображение текущего баланса (разница между доходами и расходами).
- ✅ Просмотр списка всех транзакций (расходы визуально отличаются от доходов).
- ✅ Фильтрация транзакций по категории, типу (доходы/расходы/все) и дате (день, неделя, месяц, произвольный период).
- ✅ Удаление транзакции с подтверждением.
- ✅ Редактирование транзакции.
- ✅ Сохранение данных между сессиями (localStorage).
- ✅ Демо-данные при первом запуске (3–5 примеров).
- ✅ Адаптивный интерфейс (десктоп, планшет, мобильные устройства).

## ⭐ Дополнительные возможности (желательные)

- ✅ Статистика расходов и доходов по категориям.
- ✅ Экспорт данных в JSON.
- ✅ Сброс данных к демо-набору.
- ✅ Три темы оформления: светлая, тёмная, old money.

## 🚀 Запуск проекта

### Локальный запуск

1. Склонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd finance-app
   ```

2. Откройте файл `index.html` в браузере через **Live Server** (рекомендуется) или просто дважды кликните по файлу (модули могут не работать без сервера).
